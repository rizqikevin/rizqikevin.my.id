## Overview

SuXesstories Platform Enhancement was part of my backend development internship at PT Batu Tala Saudara. The work focused on improving registration and onboarding flows for a psychotest and self-development platform, especially for job fair event usage.

## Key Contributions

- Improved the **QR code self-registration** flow for job fair participants
- Worked on frontend implementation using **Vue.js**
- Supported backend integration using **Firebase** and **Express.js**
- Helped simplify onboarding and automate registration-related workflows
- Assisted in preparing technical documentation and user guidance for the delivered features

## Outcome

- Reduced friction during event registration
- Supported a better onboarding experience for new users
- Contributed to smoother platform operation during job fair execution

## What I Learned

This project improved my understanding of:

- event-based product constraints
- user onboarding optimization
- coordination between frontend and backend implementation
- delivering practical improvements under short timelines
