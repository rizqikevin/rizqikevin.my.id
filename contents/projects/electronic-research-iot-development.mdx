## Overview

This project represents a broader body of work in electronic research and IoT system development during my time as an Electronic Researcher at PT Delameta Bilano. The scope included hardware-software integration, prototyping, diagnostics, embedded troubleshooting, and backend-connected monitoring flows.

## Key Contributions

- Contributed to the development of an automatic coffee vending machine based on microcontroller systems
- Supported integration between sensors, payment systems, and control devices
- Built and tested multiple IoT and embedded prototypes for automation and monitoring use cases
- Connected hardware workflows with backend services and databases for real-time processing
- Performed troubleshooting and validation to improve reliability before deployment

## Technical Focus

- microcontroller programming
- electronics testing and troubleshooting
- backend-connected IoT workflows
- system integration between device, server, and database
- Python-based testing support for validation flows

## Impact

- Improved prototype reliability and hardware-software integration quality
- Reduced integration bottlenecks through close coordination across disciplines
- Helped produce practical research outputs that could be implemented in real use cases
