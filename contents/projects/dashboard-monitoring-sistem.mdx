## Overview

Dashboard Monitoring Sistem is a web-based operational dashboard project developed in a professional environment at PT Delameta Bilano. The main objective was to make operational data easier to access, faster to interpret, and more reliable for daily monitoring activities.

## Key Contributions

- Built frontend modules using **React.js** and **Vue.js**
- Integrated multiple **REST API** endpoints to display live operational data
- Created reusable and responsive UI components to speed up feature delivery
- Improved rendering flow and frontend performance for smoother data access
- Collaborated with backend engineers to keep client-server synchronization stable

## Impact

- Improved development efficiency through reusable component patterns
- Increased data access speed for operational users
- Reduced friction when monitoring system activity in a browser-based dashboard

## What I Learned

This project strengthened my experience in:

- frontend architecture for dashboard-style applications
- component reuse and consistency across pages
- API-driven interface design
- debugging production-facing web applications
- balancing speed of delivery with maintainability
